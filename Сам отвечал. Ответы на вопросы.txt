1)What is a Class/Object/OOPs concepts?
Класс - пользовательский тип данных

Объект - экземпляр класса.

Концепции ооп - 
  Абстракции - не нужно подробно все описывать
  Инкапсуляция - Все разделено на уровни доступа private, pablic, protected, по умолчанию
  Полиморфизм - переиспользование кода 
  (перегрузка, переопределение методов произодного класса)
  Наследование

2)What is meant by Overloading?
  Перегрузка это когда у методов одинаковое название 
  но разный тип передаваемых аргументов или разный порядок или разное кол-во

3)What are constructors in Java?
  Конструктор это специальный метод 
  который вызывается когда выделяется память под объект нашего класса 
  посредством оператора new

4)What is a package in Java? List down various advantages of packages.
  Пакет это папка в которой хранятся наши файлы программы. 
  Используя пакеты мы можем отсортировать наши файлы по логическим модулям 
  что упростит обращение с программой. Используя пакеты 
  мы можем удобно импортировать файлы в программу

5)What is the difference between a local variable and an instance variable?
  Локальная переменная это переменная в методе. А instance variable это поле класса.
  Разница в жизненном цикле. 
  Локальная переменная перестает существовать когда мы выходим из тела метода.

6)Differentiate between the constructors and methods in Java?
  Методы могут вызываться в любом месте программы. 
  А конструктор вызывается при выделении памяти для объекта класса.

7)What is the difference between break and continue statements?
  Break выходит из цикла а continue пропускает итерацию

8)What is a constructor overloading in Java?
  Когда мы не создаем конструктора то он автоматически создается(конструктор по умолчанию). 
  Как только мы создадим конструктор то конструктор автоматически не создастся.
  Мы можем создать много конструкторов 
  и то какой конструтор вызовется определяет какие аргументы, порядок и количество
  мы передадим ему при создании объекта

9)Explain typecasting?
  Привдение типов - преобразование одного типа в другой. int в long например. 
  Можно и long в int но тогда мы рискуем все испортить бо long больше int 
  и если в long будет значение большее чем может вместить int то будет ошибка. 

10)Explain what are static methods and variables?
  Инициализация статических членов происходит при инициализации класса.
  Инициализация не статических членов происходит при создании объекта.
  Т.е. мы може обращаться к сатичесим членам класса не создавая объект класса,
  напрямую через имя класса в отличие от не статических.
  И статическая переменная одна для всех объектов класса.
